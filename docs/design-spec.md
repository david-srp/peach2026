# Peach 2026 — 设计规格文档

> 桃子小三班 AI 体验课 · 互动绘本网页
> 创建日期：2026-03-23

---

## 1. 项目概述

为桃子小三班（4岁幼儿园小朋友）打造的 AI 体验课互动绘本网页。课堂上由老师通过投屏展示，以翻页画册的形式引导孩子体验 AI 的三种核心能力：知识问答、文本生成、图像生成、视频生成。

## 2. 技术栈

- **框架**：Reveal.js（演示文稿框架）
- **部署**：静态文件 → GitHub (`david-srp`) → Vercel 自动部署
- **项目目录**：`~/Projects/peach2026/`
- **插图生成**：Gemini Image Pro（通过 LiteLLM API）
- **图片风格**：Bluey 风扁平 2D 儿童插画（详见 `docs/style-dna.md`）

## 3. 目标受众

- **主要**：4岁幼儿园小朋友（现场投屏观看）
- **次要**：老师（通过隐藏提示辅助讲课）

## 4. 视觉设计

### 4.1 Style DNA
详见 `docs/style-dna.md`，核心要素：
- Bluey (Ludo Studio) 风格扁平 2D 儿童插画
- 圆形 + 圆角矩形为主，无尖角
- 暖金色光照，彩色阴影（不用灰色）
- 水彩纸纹理质感
- 色板：橙 `#E27A37`、天蓝 `#88CAFC`、奶油 `#E4DCBD`、金黄 `#EDCC6F`
- 辅助色：薰衣草、薄荷
- 文字/深色：深靛蓝 `#404066`

### 4.2 角色系统
参考图：`docs/characters-reference.jpg`

- **豆包**：波波头短发女孩，黑色外套+红围巾，温柔微笑 — 代表 AI 助手
- **Austin**：黑色短发男孩，红色麻花毛衣 — 代表小朋友

角色在流程引导页出现，内容想象页用独立主题插图。不强制每页出现。

### 4.3 文字风格
- **极简绘本风**：每页只有 1 个重点句，大字体
- 插图丰富饱满，文字精简
- 字体：圆润可爱的中文字体（推荐站酷快乐体或类似）

### 4.4 老师提示开关
- 右上角小图标开关，默认**关闭**
- 点击后弹出半透明浮层，显示该页的老师引导话术
- 不影响主视觉，孩子不会注意到

## 5. 交互设计

### 5.1 翻页方式（三种都支持）
- 键盘方向键（左右）
- 触屏滑动
- 页面两侧箭头按钮

### 5.2 进度条
- 底部细进度条，显示当前位置

### 5.3 页面动效（中度）
- 翻页平滑过渡动画
- 每页局部惊喜动效：角色轻微弹跳、元素飘入、星星闪烁等
- **元素出场顺序 = 讲课逻辑顺序**（使用 Reveal.js fragment 功能）
  - 例：先出现问题文字 → 再弹出豆包回答卡片 → 最后飞入惊喜元素

## 7. 插图生成策略

### 7.1 生成工具
- API：LiteLLM → `gemini-3-pro-image-preview`
- Base URL：`https://litellm.vllm.yesy.dev/v1`

### 7.2 Prompt 结构
每张插图 prompt = 场景描述 + Style DNA 后缀（见 `docs/style-dna.md` 底部的 Prompt Suffix）

### 7.3 角色一致性
- 豆包和 Austin 出现时，prompt 中需要包含角色描述以保持一致
- 豆包：短发波波头女孩，穿深色外套和红围巾
- Austin：黑色短发男孩，穿红色麻花毛衣

### 7.4 图片规格
- 尺寸：16:9 横版（适配投屏）
- 格式：PNG
- 存放：`public/images/` 目录

## 8. 视频占位设计

梦想小导演章节的视频位置使用占位方案：
- Bluey 风格的静态插图作为封面
- 居中半透明播放按钮图标
- 底部标注 "视频稍后更新"
- HTML 结构预留 `<video>` 标签，`src` 属性为空，方便后续替换

## 10. 部署流程

1. Claude Code 在 `~/Projects/peach2026/` 完成开发
2. `gh repo create david-srp/peach2026 --public --source=. --push`
3. Vercel 连接 GitHub 仓库自动部署
4. 产出可访问链接

## 11. 实施分步计划

### Phase 1：项目骨架
- 初始化 Reveal.js 项目
- 配置自定义 Bluey 主题（CSS）
- 实现翻页交互（键盘+触屏+箭头）
- 实现进度条
- 实现老师提示开关组件

### Phase 2：内容填充（占位图版）
- 用彩色占位块创建全部 36 页
- 每页写好主文字、fragment 动效顺序
- 视频占位设计

### Phase 3：插图生成
- 按页面顺序批量生成插图（Gemini Image Pro + Style DNA）
- 替换占位图

### Phase 4：动效打磨
- 为每页添加局部动效
- 调整 fragment 出场顺序

### Phase 5：部署上线
- GitHub 仓库创建
- Vercel 部署
- 测试链接
